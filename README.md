# OSM data parser

Data: taipei.osm
Data format: https://wiki.openstreetmap.org/wiki/OSM_XML

# Utility Scripts
* xml_analysis.py: count number of Nodes, Way ... in the OSM file
  ```
  $ python3 xml_analysis.py

  defaultdict(<class 'int'>, {'bounds': 1, 'node': 44616, 'way': 6687, 'relation': 572})
  ```
